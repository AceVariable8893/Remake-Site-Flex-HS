# Projet de cours - Recréer le site Flex HS

Le but de ce projet était de refaire au mieux le site Flex HS. Le site ne lui même possède des contraintes techniques compliqués à reproduire.

## Contraintes :

Il fallait obligatoirement utiliser Bootstrap, et n'utiliser aucun css (le seul fichier css que l'on pouvait utiliser était pour changer les couleurs de bootstrap pour les adapter).
